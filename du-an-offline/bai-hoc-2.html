Bài học 2: Biến và kiểu dữ liệu trong C++

1. Biến trong C++ là gì?

Biến là nơi lưu trữ dữ liệu trong bộ nhớ của máy tính. Mỗi biến có một tên và kiểu dữ liệu xác định loại giá trị nó chứa.

2. Các kiểu dữ liệu cơ bản trong C++

  • int: số nguyên (ví dụ: 1, -5, 100)
  • float: số thực (ví dụ: 3.14, -0.5)
  • double: số thực có độ chính xác cao hơn float
  • char: ký tự đơn (ví dụ: 'a', 'Z')
  • bool: giá trị đúng hoặc sai (true hoặc false)

3. Cách khai báo biến

// Khai báo biến số nguyên
int age;

// Khai báo và gán giá trị cho biến
float pi = 3.14;
char letter = 'A';
bool isReady = true;

4. Ví dụ sử dụng biến

#include <iostream>

int main() {
  int age = 20;
  float height = 1.75;
  char grade = 'A';
  bool passed = true;

  std::cout << "Tuổi: " << age << std::endl;
  std::cout << "Chiều cao: " << height << std::endl;
  std::cout << "Điểm: " << grade << std::endl;
  std::cout << "Đậu hay không: " << passed << std::endl;

  return 0;
}
