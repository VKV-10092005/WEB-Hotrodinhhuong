Bài học 3: Vòng lặp trong C++

1. Vòng lặp là gì?

Vòng lặp cho phép bạn thực hiện một đoạn mã nhiều lần cho đến khi một điều kiện nào đó không còn thỏa mãn.

2. Các loại vòng lặp trong C++

  • for: vòng lặp có số lần lặp xác định trước.
  • while: vòng lặp lặp lại khi điều kiện đúng.
  • do...while: giống while nhưng chạy ít nhất 1 lần.

3. Cú pháp vòng lặp for

for (int i = 0; i < 5; i++) {
  // Thực hiện 5 lần
}

4. Ví dụ vòng lặp for

#include <iostream>

int main() {
  for (int i = 1; i <= 5; i++) {
    std::cout << "Lần lặp thứ " << i << std::endl;
  }
  return 0;
}

5. Ví dụ vòng lặp while

#include <iostream>

int main() {
  int i = 1;
  while (i <= 5) {
    std::cout << "Lần lặp thứ " << i << std::endl;
    i++;
  }
  return 0;
}
